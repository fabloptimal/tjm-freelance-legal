## 📱 Application

**TJM Freelance** est une application iOS permettant aux travailleurs indépendants de calculer leur Taux Journalier Moyen (TJM) selon leur statut juridique :

- Micro BNC (gratuit)
- Micro BIC, Portage salarial, SASU/EURL (version complète — achat unique 2,99 €)
