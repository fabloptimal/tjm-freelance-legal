Données collectées

L'application TJM Freelance ne collecte aucune donnée personnelle.

  • Aucun compte utilisateur requis
  • Aucune donnée transmise à des serveurs tiers
  • Aucun tracking, aucune analytique
  • Les calculs sont effectués localement sur votre appareil

Les paramètres de calcul (revenu souhaité, frais professionnels, rythme de travail) sont stockés uniquement sur votre appareil via les mécanismes natifs iOS (UserDefaults) et ne sont jamais transmis.

Récupération des taux URSSAF

Les taux de charges affichés dans l'application sont récupérés depuis un fichier JSON public hébergé sur GitHub (fabloptimal.github.io). Cette requête réseau est de nature purement technique : aucune donnée utilisateur n'est envoyée lors de cet appel. GitHub peut enregistrer des métadonnées de requête (adresse IP, user-agent) conformément à sa propre politique de confidentialité.

Achats intégrés

L'application propose un achat unique via Apple In-App Purchase. Les transactions sont gérées exclusivement par Apple. L'éditeur n'a accès à aucune information bancaire ou de paiement. Les données de paiement sont régies par la politique de confidentialité d'Apple.
